今天也没废
生活记录与自我激励应用

记录生活点滴，发现日常美好。一款帮助你记录每日成就、培养积极心态的生活记录应用。

项目简介
「今天也没废」是一款基于 React + TypeScript + Ant Design Mobile 开发的生活记录应用。它帮助用户记录每日的小成就、心情变化和生活点滴，通过正向反馈机制培养积极的生活态度。

应用采用闪应用（MiniApp）架构，支持在灵光平台直接运行，无需额外部署。
